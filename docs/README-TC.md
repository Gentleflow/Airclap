## 產品特點
#### 🌎 &nbsp;無需連接互聯網
- 無線局域網傳輸
- 面對面分享 
#### 🖥️ &nbsp;跨平台、跨系統、跨設備
- 支持 Mac, iOS, Windows, and Android 之間互傳
- 官網直接安裝
#### 🔮 &nbsp;現代設計
- 簡單易用的傳輸體驗
- 清晰直觀的界面和反饋
#### ⚡️ &nbsp;極速傳輸速度
- 使用局域網最大傳輸值
- 穩定的傳輸過程
- 實時的傳輸進度顯示
#### 📃 &nbsp; 清晰的傳輸記錄
- 豐富的快捷操作
- 直觀的文件預覽
